## Tech Stack Used
 - React js
 - Vite js
 - React router dom
 - Zod
 - React hook form
 - Tailwind css
 - vitest for testing
 - Typescript


 ## Pages

 - There are two pages
 - Login Page
 - DashBoard Page

### You can find the tests in   `/src/tests` folder

 ### Run locally

 ```bash
   pnpm install
   pnpm run dev
   
   // to run test
   pnpm run test
 ```

### Description
- Able to login
- Able to edit the data
- Able to select the data
- Pagination
- Data validation
